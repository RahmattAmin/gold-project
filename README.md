# How To run

**Install your dependency by:**

```bash
npm i
```

# Migrate and Seed your file

**Migrate your Database**

```bash
npm run migrate
```

**Seed your Database**..."if needed\*

```bash
npx knex seed:run
```

# Run project by:

```bash
npm run dev
```
